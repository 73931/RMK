# RMK
RimWorld Mod Korean Project

림월드 모드 한국어 번역 통합 프로젝트입니다.

번역팩을 Github에서 직접 다운로드 받으려면 Code-Download ZIP 버튼을 누르세요.<br>
다운받은 파일은 림월드 로컬 모드 폴더에 풀어서 사용하세요.<br>
<br>
