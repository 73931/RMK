GitHub를 통해서 번역 추가 및 수정을 하고 싶으신 분은 파일을 이 폴더에 정리해서 Pull Request를 보내주시기 바랍니다.
그 외는 림월드 마이너 갤러리를 통해 공유해주시기 바랍니다.


[RMK와 협업을 위한 번역자 안내사항]

1. 한 모드의 파일은 한 폴더에 정리해주세요.

2. 모드를 새로 추가하는 경우 스팀 창작마당 주소 및 PackageID를 텍스트 파일 등에 함께 기재해주세요.

3. PackageID는 가독성을 위해서 대문자를 살려주기 바랍니다.
	예를 들어, RimPy 화면에서는 모든 PackageID가 소문자로 변환되어 보여집니다.
	각 모드의 About.xml 파일에서 실제 PackageID를 확인할 수 있습니다.

4. 림추출기로 추출할 때 가능한 xlsx 파일로 추출하기 바랍니다.
	xml로 추출한 뒤 xlsx로 변환하면 원문 정보가 소실됩니다.
	원문도 함께 있어야 추후 검수 또는 비교가 수월합니다.

5. xml 파일의 유효성을 담보하기 위해 xml 파일을 직접 수정하지 말고 림추출기를 사용해 xlsx 파일을 통하여 변환하는 것이 좋습니다.
	xml 파일을 직접 수정하는 경우 휴먼 에러로 인한 xml 오류가 꽤 자주 발생합니다.

6. 번역을 수정하기 전에 해당 모드 폴더에 xlsx 파일이 존재하는지 먼저 확인해주세요.
	번역하고자 하는 부분이 해당 xlsx 파일에 기재된 노드를 수정하는 것 만으로 해결되는 경우 가능한 그렇게 해주시기 바랍니다.
	수정한 내용을 색깔 등으로 표시해주면 더욱 좋습니다.
	불필요하게 추출을 갱신하는 경우 기존 번역과 무엇이 다른지 확인하기 위한 검수 소요가 늘어나게 됩니다.
