GitHub를 통해서 번역 추가 및 수정을 하고 싶으신 분은 파일을 이 폴더에 정리해서 Pull Request를 보내주시기 바랍니다.

[작업시 유의사항]
1. 한 모드의 파일은 한 폴더에 정리해주세요.
2. 모드를 새로 추가하는 경우 스팀 창작마당 주소 및 PackageID를 텍스트 파일 등에 함께 기재해주세요.
3. PackageID는 가독성을 위해서 대문자를 살려주기 바랍니다. 예를 들어, RimPy 화면에서는 PackageID가 소문자로 변환되어 보여집니다. 각 모드의 About.xml 파일에서 PackageID를 확인할 수 있습니다.
4. 림추출기로 추출할 때 가능한 xlsx 파일로 추출하기 바랍니다. (xml 파일로 추출 후 xlsx로 변환 X) 원문이 살아있어야 추후 검수 또는 비교가 수월합니다.
5. xml 파일의 유효성을 담보하기 위해 xml 파일을 직접 수정하지 말고 림추출기를 사용해 xlsx 파일을 통하여 변환하기 바랍니다. xml 파일을 직접 수정하는 경우 실수로 인해 xml 오류가 자주 발생합니다.
